# Automated Mission for Online Graphs Utility System (AMOGUS)

A simple auto-discovery program that you can use to transfer files between machines coded in C!

## Instructions

### Compilation
```bash
gcc main.c -o amogus
```

### Receive a file

```bash
./amogus --receive <file-path>
```

### Send a file

```bash
./amogus --send <file-path>
```
